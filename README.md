# Zero

A config-driven backend generator that eliminates repetitive code by automatically creating scalable APIs, CRUD operations, and core backend structure from simple configuration.

## 🛠 Tech Stack

- TypeScript
- Handlebars
- JavaScript
- PostgreSQL
- Express

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/Rajpatel148/Zero
cd Zero
```

Install dependencies:

```bash
npm install zero-backend
```

## ▶️ Usage

```bash
zero generate -c <YAML file path>
```
